# SynapseOS

### Personal Data Operating System
AI system that understands and connects personal knowledge

---

## Overview

SynapseOS is a **Personal Data OS** that helps users organize, search, and connect their knowledge across different sources using AI.

It acts like a **second brain**, enabling:
- Semantic search
- Knowledge linking
- Intelligent insights

---

## Problem

People store data everywhere — notes, bookmarks, chats — but:
- Information is fragmented
- Hard to retrieve
- No meaningful connections

---

## Solution

SynapseOS creates a **central AI-powered system** that:
- Stores structured + unstructured data
- Uses embeddings for semantic understanding
- Builds relationships using graph databases

---

## Features

-  Authentication system
-  Personal knowledge storage
-  Semantic search (RAG-based)
-  AI-powered insights
-  Knowledge graph connections
-  Fast API responses

---

## 🛠 Tech Stack

### Frontend
- React.js (TypeScript)

### Backend
- Node.js (TypeScript)
- Express.js

### AI Service
- Python (FastAPI)
- Embeddings + RAG

### Database
- PostgreSQL → structured data
- Chroma → vector database
- Neo4j → knowledge graph
